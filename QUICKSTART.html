üé® Customization Guide

Change Event Date

Edit the countdown date in js/main.js (line ~6):

const eventDate = new Date('2026-03-30T00:00:00').getTime();

Change 2026-03-30 to your actual event date.

Update Colors

Edit the color scheme in index.html Tailwind config:

colors: { "primary": "#25c0f4", // Cyan blue "secondary": "#a855f7", // Purple "background-dark": "#0a0f12" // Dark navy }

Modify Text Content

Edit directly in index.html:

  • Update the main heading
  • Change description text
  • Modify navigation labels
  • Update footer information

Add Navigation Links

Edit the navigation section to link to actual pages or sections.
